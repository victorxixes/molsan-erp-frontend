# MOLSAN ERP — Glass Luxe 2027

Sistema ERP moderno para la gestión de firmas notariales, diseñado con arquitectura modular, IndexedDB, interfaz premium y rendimiento optimizado para grandes volúmenes de datos.

---

## 🚀 Características principales

- **Dashboard dinámico** con KPIs en tiempo real  
- **Listado avanzado** con filtros, paginación y búsqueda  
- **Importación de Excel** con normalización automática  
- **IndexedDB** como motor de almacenamiento local  
- **Backup & Restore** completos en JSON  
- **Informes Premium** con gráficos (Chart.js)  
- **Permisos de usuario** con control granular  
- **Sidebar Glass Luxe 2027** con estado persistente  
- **Arquitectura modular** y escalable
